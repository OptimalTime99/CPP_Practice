## [의사코드]
1. 플레이어의 HP를 초기값으로 설정한다
2. 몬스터의 공격력을 데미지 값으로 사용한다
3. 현재 HP에서 데미지를 뺀 값을 계산하고 미리보기를 표시한다
4. 미리보기 이후 실제 HP는 변경되지 않았음을 확인한다
5. HP를 초기화한다
6. 데미지 값이 저장된 주소를 이용하여 실제 HP를 감소시킨다
7. 감소된 HP를 출력하여 변화가 있음을 확인한다
8. HP를 초기화한다
9. 데미지 값을 직접 참조하여 HP를 감소시킨다
10. 변화된 HP를 출력한다
11. 세 방식의 차이를 정리한다

## [필요한 데이터 정리]
- 플레이어 현재 HP
- 몬스터 공격력(= 데미지)
- 플레이어 미리보기 HP

## [필요한 함수 정리]
- 플레이어의 HP를 얻는 함수
- 플레이어의 HP 감소 미리보기 함수: CallByValue
- 플레이어의 HP가 감소하는 함수: CallByPointer, CallByReference
- 플레이어의 현재 HP를 출력하는 함수
- 플레이어의 미리보기 HP를 출력하는 함수
- 이외의 텍스트를 출력하는 함수
